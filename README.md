## :rocket: Getting Started

To experiment with the code in this repository, follow these steps:

1. Install prerequisites

    - [Git](https://git-scm.com/downloads) - Version Control System
    - [Python](https://www.python.org/downloads/) - Programming Language
    - [uv](https://docs.astral.sh/uv/getting-started/installation/) - Python Package and
  Project Manager

1. Clone the repository

    ```bash
    git clone https://github.com/althaf-07/numerorum-primorum.git
    ```

1. Navigate to the project directory

    ```bash
    cd numerorum-primorum
    ```

1. Update the project's environment

    ```bash
    uv sync
    ```

1. Experiment with the code in this [Python script](./src/numerorum_primorum/main.py)
